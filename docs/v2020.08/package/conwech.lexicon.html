Prefixes for all zillion period values. Indexing this tuple with a zillion period value will return the appropriate Conway-Wechsler prefix (everything before the “illi”/”illion”).

All prefixes are constructed using the appropriate sub-prefix combinations and exception rules defined by the Conway-Wechsler naming system.

The first prefix is ‘n’ (for building ‘nilli’ components) and the next nine prefixes are based on the standard number names adopted before the inception of the Conway-Wechsler system.

Examples

>>> from conwech.lexicon import ZILLION_PERIOD_PREFIXES
>>> ZILLION_PERIOD_PREFIXES[:10]
('n', 'm', 'b', 'tr', 'quadr', 'quint', 'sext', 'sept', 'oct', 'non')
>>> tuple(prefix + 'illon' for prefix in ZILLION_PERIOD_PREFIXES[1:5])
('million', 'billion', 'trillion', 'quadrillion', 'quintillion')
>>> ZILLION_PERIOD_PREFIXES[12]
'duodec'
>>> ZILLION_PERIOD_PREFIXES[345]
'quinquaquadragintatrecent'
>>> 'illi'.join(ZILLION_PERIOD_PREFIXES[int(p)] for p in '12,000,345'.split(','))) + 'illion'
'duodecillinilliquinquaquadragintatrecentillion'
>>> ZILLION_PERIOD_PREFIXES[106]
'sexcent'
>>> ZILLION_PERIOD_PREFIXES[600]
'sescent'
